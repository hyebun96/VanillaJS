# Variable

#### 변수는 
1. Create
2. Initialize
3. Use

#### Variable(변수)의 종류
+ var(variavle 변수)
    + 변수를 선언
    + 변수를 값으로 초기화 가능
+ let
    + 블록 범위 지역 변수를 선언
    + 변수를 값으로 초기화 할 수 있음
    + const와 대조해서 사용
+ const(constant 상수)
    + 읽기 전용 유명(named) 상수를 선언
    + 변수를 초기화 할 수 없음
    + 초기화 시키지 않기 위해서 사용
    + 즉, 변화를 주고 싶지 않을 떄 사용
