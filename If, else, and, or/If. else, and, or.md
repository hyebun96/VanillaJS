# If, else, and, or

### If
~~~
if(condition){
    block
} else {
    block
}
~~~
+ If, else는 기본적으로 프로그래밍에서 어떻게 모든게 잘 동작하는지 보여주는 것
+ 조건이란 뭐든 참을 의미
+ 피연산자란 다른 조건들을 합치는것
+ && : and
+ || : or

### If 조건문 유형
true && true = true; <br/>
false && true = false; <br/>
true  && false = false; <br/>
false && fasle = fasle; <br/>

true || true = true; <br/>
false || true = false; <br/>
true  || false = false; <br/>
false || fasle = fasle; <br/>
